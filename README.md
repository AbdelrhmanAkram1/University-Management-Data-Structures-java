# University Management System (DSA-Based)

A robust **Java Desktop Application** designed to manage university operations efficiently using core **Data Structures and Algorithms (DSA)**. This project focuses on implementing custom data structures to handle student records, course registrations, and system actions.

---

##  Features

* **Student Management**: Add, update, and track student information using custom linked nodes.
* **Course Enrollment**: Manage courses and link them to students dynamically.
* **Action History**: Implements a **Stack** data structure to track recent actions (useful for Undo/Redo logic).
* **Interactive GUI**: User-friendly interface built with Java Swing/AWT for seamless interaction.
* **Optimized Data Handling**: Uses efficient searching and sorting logic based on DSA principles.

##  Tech Stack

* **Language**: Java
* **Framework**: Java Swing (GUI)
* **Data Structures**: 
    * Linked Lists (Custom Nodes)
    * Stacks (for Action Tracking)
    * Dynamic Arrays

##  Project Structure

* `src/`: Contains all `.java` source code files.
    * `Models/`: Data structures like `StudentNode`, `CourseNode`, etc.
    * `GUI/`: Frame and Panel classes for the interface.
    * `Logic/`: The controller logic for managing data flow.
* `bin/`: Compiled bytecode (optional to include in repo).
